## Установка:

- Получить api_id и api_hash на https://my.telegram.org/apps
- Запустить скрипт:

### Windows:
Запустите run.bat дважды щелкнув на файл или выполнив команду в командной строке Windows:
```
run.bat
```

### Linux:
```
chmod +x run.sh
```
```
./run.sh
```
